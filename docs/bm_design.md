## 3. 상세 BM 설계

### 3.1 차량/스킨 BM

#### 무료 구조
- 스토리 진행으로 기본 차량 3종 획득
- 업적 달성으로 추가 차량 2종 해금
- 이벤트에서 한정 차량 획득 기회

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 차량 해금 | 개별 차량 구매 | $2.99~$6.99 |
| 차량 번들 | 차량 3종 + 스킨 | $9.99 |
| 차량 스킨 | 외형 변경 (성능 동일) | $1.99~$4.99 |
| 한정 차량 | 시즌 한정 판매 | $6.99~$9.99 |

### 3.2 능력 카드 BM

#### 무료 구조
- 레이스 클리어로 기본 카드 획득
- 일일 무료 뽑기 1회
- 업적으로 카드 조각 파밍

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 카드 뽑기 | 10연차 1회 | 프리미엄 재화 2,000 |
| 픽업 배너 | 특정 카드 확률 UP | 동일 |
| 천장 시스템 | 50연차 SSR 확정 | - |
| 카드 스킨 | 카드 외형 변경 | $0.99~$1.99 |

### 3.3 튜닝/성장 BM

#### 무료 구조
- 레이스 보상으로 튜닝 재료 획득
- 기본 튜닝 무료
- 일일 골드 레이스 무료

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 튜닝 패키지 | 튜닝 재료 대량 | $2.99~$6.99 |
| 즉시 튜닝 | 튜닝 시간 스킵 | 프리미엄 재화 |
| 파츠 패키지 | 고급 파츠 세트 | $4.99~$9.99 |

### 3.4 광고 BM

| 광고 유형 | 적용 | 보상 |
|----------|------|------|
| 리워드 광고 | O | 추가 보상, 무료 뽑기, 부스터 |
| 인터스티셜 | 제한적 | 5레이스마다 (스킵 가능) |
| 광고 제거 패스 | O | $2.99 (영구) |

## 6. BM 밸런스 원칙

### P2W 방지선
- 모든 트랙 무과금 클리어 가능
- 차량/카드 성능보다 실력이 더 중요
- 튜닝 격차는 시간으로 해소 가능

### 과금 압박 최소화
- 레이스 횟수 무제한
- 짧은 세션으로 광고 피로 최소화
- 천장 시스템으로 뽑기 좌절 방지
