## 1. 화면 설정

| 항목 | 값 |
|------|-----|
| **방향** | **Landscape (가로)** |
| **기준 해상도** | 1920 × 1080 |
| **지원 비율** | 16:9 ~ 21:9 |
| **타겟 FPS** | 60fps |

### 방향 선택 이유
- 레이싱 트랙 시야 확보 필수
- 좌우 조향 양손 조작
- 레이싱 게임 표준 가로 모드

## 4. 터치 영역

| 동작 | 영역 | 제스처 |
|------|------|--------|
| 좌회전 | 좌측 하단 | 홀드 |
| 우회전 | 우측 (조향) | 홀드 |
| 브레이크 | 중앙 하단 | 탭/홀드 |
| 부스터 | 우측 부스터 | 탭 |
| 스킬 사용 | 스킬 버튼 | 탭 |
| 아이템 사용 | 아이템 버튼 | 탭 |

### 터치 피드백
- 조향: 차량 기울기 반영
- 부스터: 화면 흔들림 + 속도선
- 아이템: 획득/사용 이펙트
- 충돌: 화면 쉐이크
