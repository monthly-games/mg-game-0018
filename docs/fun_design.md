## 2. 코어 루프 (Core Loop)

```
┌─────────────────────────────────────────────────────────┐
│  1. 레이스 참가                                          │
│         ↓                                               │
│  2. 레이스 중 능력 카드 사용                             │
│         ↓                                               │
│  3. 순위에 따른 보상 획득                                │
│         ↓                                               │
│  4. 차량 튜닝/카드 강화 ─────────────────→ (1로 반복)     │
└─────────────────────────────────────────────────────────┘
```

### 루프 상세 설명

1. **레이스 참가**: 다양한 트랙/모드에서 레이스
2. **능력 카드 사용**: 레이스 중 전략적 카드 사용
3. **보상 획득**: 순위에 따른 자원/카드 보상
4. **튜닝/강화**: 차량 성능 업그레이드, 카드 덱 강화

## 3. 재미 축 (Fun Pillars)

### 3.1 스피드/피드백 (Speed)
- **목표**: 빠른 속도감과 짜릿한 피드백
- **핵심 경험**: 부스터 터지며 질주하는 쾌감
- **주요 요소**:
  - 스피드 이펙트
  - 드리프트 시스템
  - 부스터 메커니즘
  - 충돌/회복 피드백

### 3.2 튜닝/커스터마이징 (Customization)
- **목표**: 나만의 차량을 만드는 재미
- **핵심 경험**: 내 차가 최고! 하는 자부심
- **주요 요소**:
  - 파츠 업그레이드
  - 외형 커스터마이징
  - 스티커/데칼
  - 특수 효과

### 3.3 카드/능력 관리 (Build)
- **목표**: 레이스 전략을 카드로 설계하는 재미
- **핵심 경험**: 이 카드 조합으로 역전! 하는 쾌감
- **주요 요소**:
  - 능력 카드 덱
  - 카드 시너지
  - 상황별 전략
  - 카드 수집

## 4. 디자인 레버 (Design Levers)

### 4.1 짧은 레이스 루프
| 레버 | 설명 |
|------|------|
| 1~2분 레이스 | 짧은 레이스(1~2분) 중심으로 루프 설계 |
| 구현 방향 | 3바퀴 기준 90초, 스프린트 모드 60초, 마라톤 모드 3분 |

### 4.2 카드-레이스 연결
| 레버 | 설명 |
|------|------|
| 튜닝/카드 영향 | 튜닝/능력 카드가 레이스 스타일에 큰 영향(드리프트, 부스터 등) |
| 구현 방향 | 드리프트 특화 빌드, 부스터 특화 빌드, 방해 특화 빌드 |

### 4.3 고스트 경쟁
| 레버 | 설명 |
|------|------|
| 리플레이/고스트 | 레이스 리플레이/고스트 경쟁 제공 |
| 구현 방향 | 개인 최고 기록 고스트, 친구 고스트, 글로벌 랭커 고스트 |

## 5. 재미 측정 지표 (KPI)

| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| 일일 레이스 횟수 | 15회 이상 | 레이스 로그 |
| 평균 레이스 시간 | 90초 내외 | 레이스 시작→종료 |
| 튜닝 변경 빈도 | 주 5회 이상 | 튜닝 변경 로그 |
| 카드 수집률 | 월 50% 이상 | 도감 달성률 |
| D7 리텐션 | 30% 이상 | 7일차 잔존율 |
