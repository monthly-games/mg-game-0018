## 3. 1년차 라이브 플랜

### 3.1 초기 (1~3개월차)

| 주차 | 주요 활동 |
|------|----------|
| 1~2주 | 소프트 런칭, 버그 수정 |
| 3~6주 | 기본 레이싱/RPG 밸런스 |
| 7~10주 | 튜닝 시스템 튜토리얼 강화 |
| 11~12주 | 첫 시즌 레이싱 준비 |

**핵심 과제:**
- 레이싱+RPG 밸런스
- 튜닝 시스템 UX
- 초반 재미 밸런스

### 3.2 중기 (4~8개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 4개월 | 주간 타임어택 시작 |
| 5개월 | 시즌별 레이싱 테마 (도시) |
| 6개월 | 고스트 레이스 시스템 |
| 7개월 | 시즌별 레이싱 테마 (사막) |
| 8개월 | 여름 그랑프리 이벤트 |

**핵심 과제:**
- 타임어택 사이클 정착
- 시즌 테마 레이싱 검증
- 고스트 레이스 효과 분석

### 3.3 후기 (9~12개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 9개월 | 차량/캐릭터 스킨 이벤트 |
| 10개월 | 시즌별 레이싱 테마 (눈) |
| 11개월 | 그랑프리 챔피언십 예선 |
| 12개월 | 1주년 그랑프리 본선 |

**핵심 과제:**
- 스킨 BM 검증
- 챔피언십 시스템
- 장기 유저 리텐션
