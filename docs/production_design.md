## 1. 재미 요소 (Fun Factors)

### 1.1 핵심 재미 (Core Fun)
| 요소 | 설명 | 강도 |
|------|------|------|
| **레이싱** | 카툰 스타일의 스피드감 있는 경주 | ★★★★★ |
| **능력 카드** | 전략적 카드 사용으로 경기 역전 | ★★★★★ |
| **차량 성장** | RPG식 차량 업그레이드 | ★★★★☆ |
| **커스터마이징** | 차량 꾸미기 | ★★★☆☆ |

### 1.2 만족감 요소 (Satisfaction)
- **역전 승리**: 카드 사용으로 극적인 역전
- **신기록**: 트랙 최고 기록 갱신
- **차량 업그레이드**: 스탯 증가의 성장감
- **새 트랙 해금**: 다양한 환경의 새 트랙

### 1.3 도전 요소 (Challenge)
```
도전 구조:
├── 트랙별 최적 카드 조합
├── 타이밍 기반 부스트
├── PvP 실시간 경쟁
├── 리그 시스템
└── 타임어택 모드
```

### 1.4 사회적 요소 (Social)
- **실시간 PvP**: 실시간 멀티플레이어 레이싱
- **비동기 PvP**: 고스트 기록 대전
- **리더보드**: 트랙별/시즌별 랭킹
- **길드 레이스**: 팀 레이싱 대회

## 2. 게임 루프 (Game Loop)

### 2.1 코어 루프 (3~5분)
```
┌─────────────────────────────────────────────────┐
│                    코어 루프                      │
├─────────────────────────────────────────────────┤
│  [카드 장착] → [레이스 시작] → [카드 사용]        │
│       ↑                            ↓             │
│       └──── [보상 획득] ← [레이스 종료] ←┘       │
└─────────────────────────────────────────────────┘
```

**단계별 상세:**
1. **카드 장착** (15초): 레이스용 카드 3장 선택
2. **레이스 시작** (5초): 트랙 로딩, 카운트다운
3. **카드 사용** (레이스 중): 타이밍에 맞춰 카드 발동
4. **레이스 종료** (5초): 순위 결정
5. **보상 획득** (5초): 코인, 경험치, 카드

### 2.2 세션 루프 (5~15분)
```
[접속] → [방치 보상] → [일일 퀘스트] → [레이스] → [차량 업그레이드]
                                            ↓
                              [카드 강화] ← [새 카드 획득]
```

### 2.3 메타 루프 (장기)
```
┌──────────────────────────────────────────────────────┐
│                     메타 진행                         │
├──────────────────────────────────────────────────────┤
│  [레이스 승리] → [코인/경험치] → [차량 강화]          │
│         ↓                              ↓             │
│  [새 트랙 해금]                  [리그 승급]          │
│         ↓                              ↓             │
│  [고급 차량/카드] ←─────────────────────┘            │
└──────────────────────────────────────────────────────┘
```

### 2.4 플로우 다이어그램
```
[앱 시작]
    ↓
[메인 화면] ─────→ [차고]
    │              [카드 컬렉션]
    │              [상점]
    │              [설정]
    ↓
[레이스] → [트랙 선택] → [카드 장착] → [경주] → [결과]
```

## 3. BM 모델 (Business Model)

### 3.1 수익화 전략
| 채널 | 비중 | 설명 |
|------|------|------|
| **리워드 광고** | 40% | 연료 충전, 보상 2배 |
| **차량/카드 팩** | 30% | 프리미엄 차량/카드 |
| **시즌 패스** | 20% | 시즌 전용 보상 |
| **편의 기능** | 10% | 연료, 즉시 업그레이드 |

### 3.2 과금 포인트
```
비과금 유저 경험:
├── 모든 트랙 플레이 가능
├── 기본 차량/카드 제공
├── 연료 시간 회복 (10분당 1)
└── 광고 시청으로 보상 획득

과금 유저 혜택:
├── 광고 제거 ($2.99)
├── 프리미엄 차량 ($2.99~$9.99)
├── 다이아 패키지 ($0.99~$49.99)
├── 시즌 패스 ($9.99/시즌)
└── 무한 연료 ($4.99/주)
```

### 3.3 가격 정책
| 상품 | 가격 | 가치 |
|------|------|------|
| 소량 다이아 (100) | $0.99 | 카드 팩 1개 |
| 중량 다이아 (550) | $4.99 | 20% 보너스 |
| 대량 다이아 (1200) | $9.99 | 30% 보너스 |
| 프리미엄 차량 | $2.99~9.99 | 개별 차량 |
| 시즌 패스 | $9.99/시즌 | 전용 차량 + 보상 |

### 3.4 리텐션 메커니즘
- **일일 보상**: 로그인 보너스
- **일일 퀘스트**: 레이스 N회, 승리 N회
- **시즌 리그**: 30일 시즌, 랭크 보상
- **시즌 패스**: 레벨업 보상

## 4. 레벨 디자인 (Level Design)

### 4.1 난이도 곡선
```
AI 강도
  ▲
  │                              ╭──────
  │                         ╭────╯
  │                    ╭────╯
  │               ╭────╯
  │          ╭────╯
  │     ╭────╯
  │╭────╯
  └──────────────────────────────────────→ 리그
    브론즈  실버  골드  플래  다이아  마스터
```

### 4.2 콘텐츠 진행
| 리그 | 해금 조건 | 새로운 요소 |
|------|----------|-------------|
| 브론즈 | 시작 | 기본 트랙 3개, 튜토리얼 |
| 실버 | 브론즈 승급 | 중급 트랙, 카드 시스템 |
| 골드 | 실버 승급 | 고급 트랙, PvP 해금 |
| 플래티넘 | 골드 승급 | 특수 트랙, 팀 레이스 |
| 다이아몬드 | 플래티넘 승급 | 전설 트랙 |
| 마스터 | 다이아 승급 | 마스터 전용 콘텐츠 |

### 4.3 밸런싱 공식
```dart
// 차량 성능
double calculateSpeed(Vehicle vehicle) {
  double base = vehicle.baseSpeed;
  double upgrade = 1 + vehicle.speedLevel * 0.02;
  double cardBoost = getActiveCardBoost();
  return base * upgrade * cardBoost;
}

// 카드 효과
double cardEffect(Card card, int upgradeLevel) {
  double base = card.baseEffect;
  double levelBonus = 1 + upgradeLevel * 0.1;
  return base * levelBonus;
}

// 레이스 점수
int raceScore(int position, int lapTime, Vehicle vehicle) {
  int positionScore = (6 - position) * 100;
  int timeBonus = max(0, 180 - lapTime) * 10;
  return positionScore + timeBonus;
}
```

### 4.4 보상 체계
| 보상 유형 | 획득 조건 | 수량 |
|-----------|----------|------|
| 레이스 코인 | 완주 | 순위별 50~300 |
| 경험치 | 완주 | 순위별 10~50 |
| 카드 드롭 | 완주 (확률) | C 70%, R 25%, SR 5% |
| 리그 보상 | 시즌 종료 | 티어별 |
| 방치 보상 | 시간 경과 | 시간당 코인 |

## 5. UI/UX 설계

### 5.1 정보 계층
```
레벨 1 (항상 표시):
├── 코인/다이아
├── 연료
├── 현재 리그
└── 알림 배지

레벨 2 (상호작용 시):
├── 차량 스탯
├── 카드 정보
├── 트랙 정보
└── 레이스 HUD

레벨 3 (팝업):
├── 레이스 결과
├── 카드 획득
├── 업그레이드 완료
└── 설정 메뉴
```

### 5.2 네비게이션 흐름
```
[메인 화면]
    ├── [레이스] → [트랙 선택] → [경주] → [결과]
    ├── [차고] → [차량 목록] → [업그레이드]
    ├── [카드] → [카드 목록] → [강화/장착]
    ├── [상점] → [차량/카드/다이아]
    └── [설정] → [옵션]
```

### 5.3 피드백 시스템
| 이벤트 | 시각 | 청각 | 촉각 |
|--------|------|------|------|
| 부스트 | 속도 이펙트 | 부스트음 | 진동 (중) |
| 카드 사용 | 카드 이펙트 | 발동음 | 진동 (약) |
| 충돌 | 충격 이펙트 | 충돌음 | 진동 (강) |
| 1등 | 우승 연출 | 팡파레 | 진동 (강) |
| 신기록 | 신기록 배너 | 신기록음 | 진동 (중) |

### 5.4 접근성
- **간단한 조작**: 좌우 스와이프 + 탭
- **자동 레이스**: 자동 진행 옵션
- **색맹 모드**: 카드 아이콘 패턴
- **폰트 크기**: 3단계 조절
- **가로/세로**: 화면 방향 선택
